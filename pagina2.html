Celulares

Son dispositivos portátiles que requieren de los servicios de telecomunicaciones para que funcionen adecuadamente. Las nuevas generaciones de celulares no sólo necesitan el servicio de telefonía móvil, sino también Internet para tener un mejor desempeño.

Internet

Es un servicio de datos tradicional de banda ancha que sólo permite tener acceso en el dominio que se contrató. Este tipo de servicio se caracteriza por manejar altas velocidades de Internet, no tener límites de transferencia para enviar y recibir datos.

Televisión por internet

Es un nuevo servicio de televisión que maneja una tarifa mensual y que se conoce como video bajo demanda. Así, el suscriptor tendrá acceso a un catálogo con diversos contenidos (películas, series), los cuales, se podrán visualizar en el momento que se deseé.

Planes tarifarios

Es un sistema tarifario de telefonía móvil, adecuado para los usuarios que requieren un servicio activo o que tienen consumos constantes del servicio.

Paquetes de Televisión

Es un servicio de televisión digital por suscripción que es proporcionado vía satelital o por cable para dar acceso a una o varias programaciones exclusivas para todo tipo de entretenimiento a cambio de que el suscriptor pague periódicamente el costo del paquete contratado.

Recargas telefonía móvil prepago

Es un sistema de prepago de telefonía móvil que se caracteriza por cobrar por adelantado el servicio de telecom. Este tipo sistema de prepago es ideal para los usuarios que tienen consumos bajos o que no desean pagar una renta, ya que permite realizar recargas a su línea de telefonía móvil.
